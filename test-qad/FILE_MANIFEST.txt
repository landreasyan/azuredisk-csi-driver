USAGE RECOMMENDATIONS
=====================

GETTING STARTED:
1. Read SETUP_COMPLETE.md (2 minutes)
2. Review QUICK_START.md (3 minutes)
3. Ensure REQUIREMENTS.md prerequisites are met (2 minutes)
4. Run: ./qad-test-helper.sh start (1 minute)

DURING TESTING:
1. Watch: ./qad-test-helper.sh logs -f
2. Monitor progress and system health
3. Run at least 10 cycles for meaningful statistics

AFTER TESTING:
1. Collect: ./qad-test-helper.sh collect /tmp/results
2. Analyze: ./qad-test-helper.sh metrics /tmp/results
3. Report: python3 qad_report_generator.py metrics.json
4. Review: Open qad-report.html in web browser

├─ Output: qad-report.html
